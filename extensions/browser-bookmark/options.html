StarNav 一键收藏设置

配置导航站地址和 Bearer Token 后，插件即可在任意网页一键保存书签到 StarNav。

连接配置

Token 获取方式

  1. 先登录 StarNav 后台，进入「Token 管理」页签。
  2. 点击「一键生成浏览器插件 Token」，或手动创建一个包含 write scope 的 Token。
  3. 复制创建后显示的 nav_xxx。Token 明文只会显示一次，请及时保存。
  4. 回到本页，把 nav_xxx 粘贴到 Bearer Token 输入框并保存设置。

开发者也可以在已登录后台的浏览器控制台中调用 POST /api/tokens 创建 Token；普通用户建议优先使用后台 Token 管理界面。

生产建议：为浏览器插件单独创建 Token；如果设备丢失，可在后台撤销该 Token。

权限说明

插件只把 Token 保存在浏览器 chrome.storage.sync 中；请求写入书签时通过 Authorization: Bearer 发送给你配置的 StarNav 站点。
