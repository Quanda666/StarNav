# StarNav 一键收藏浏览器插件

这是一个浏览器插件，可以帮助你将当前正在浏览的网页一键收藏到你的 StarNav 导航站。

## 功能特性

- **一键收藏**：自动读取当前标签页的标题和 URL。
- **信息抓取**：支持从目标网页抓取描述和 Logo。
- **智能推荐**：可调用 StarNav 后端的 AI 能力，推荐合适的分类和标签。
- **重复检测**：在保存前检测书签是否已存在。
- **灵活配置**：支持设置默认分类、标签和书签可见性。
- **Token 鉴权**：通过 Bearer Token 与你的 StarNav 站点安全通信，不依赖 Cookie。

## 安装方法

本插件尚未发布到官方商店，你需要通过开发者模式手动加载。

**以 Chrome / Edge 浏览器为例：**

1.  打开浏览器，进入“扩展”管理页面（地址栏输入 `chrome://extensions` 或 `edge://extensions`）。
2.  在页面右上角，打开“**开发者模式**”开关。
3.  点击“**加载已解压的扩展程序**”按钮。
4.  在弹出的文件选择框中，选择本插件的目录（即 `browser-bookmark` 这个文件夹）。
5.  加载成功后，你应该能在浏览器工具栏看到 StarNav 的图标。

## 配置步骤

在使用插件前，你需要先进行配置，让它知道你的 StarNav 站点地址和访问凭据（Token）。

1.  **获取 Bearer Token**
    -   登录你的 StarNav 网站后台。
    -   进入“**后台管理**”页面。
    -   切换到“**Token 管理**”页签。
    -   点击“**一键生成浏览器插件 Token**”按钮。
    -   页面会显示一个新生成的 Token，点击“**复制 Token**”按钮将其复制到剪贴板。**请注意：这个 Token 只会完整显示一次，请立即复制并妥善保管。**

2.  **配置插件**
    -   在浏览器工具栏点击 StarNav 插件图标，在弹出的窗口中点击右上角的**设置图标**（一个调整参数的图标）。
    -   或者，你也可以右键点击工具栏上的插件图标，选择“**选项**”。
    -   在设置页面中：
        -   **StarNav 地址**：填写你的 StarNav 网站首页地址，例如 `https://nav.example.com`。
        -   **Bearer Token**：粘贴你刚刚从后台复制的 Token。
    -   点击“**保存设置**”按钮。
    -   建议点击“**测试连接**”和“**刷新分类/标签缓存**”按钮，确保一切正常。状态栏会提示连接和同步结果。

## 使用说明

1.  **打开插件**：在想要收藏的网页上，点击浏览器工具栏的 StarNav 图标。
2.  **填写信息**：插件会自动填充网页的标题和 URL。你可以按需修改。
3.  **辅助功能**：
    -   **自动抓取**：点击可尝试抓取网页的描述和 Logo。
    -   **检测重复**：检查该 URL 是否已存在于你的书签中。
    -   **推荐分类/标签**：使用 AI 为当前书签推荐合适的分类和标签。
4.  **保存书签**：点击“**保存书签**”按钮即可。如果检测到重复，你可以选择“**仍然强制保存**”。
